# Arknights Character Profiles — 明日方舟角色设定集

基于 [ArknightsGameData](https://github.com/Kengxxiao/ArknightsGameData) 的游戏数据，为明日方舟全角色生成的**独立角色设定档案**。

每个 `.json` 文件就是一个角色的完整人生故事——包含了性格、经历、记忆、人际关系、语音台词、官方档案原文，以及个人故事。

> 这不是游戏数据导出。这是**角色的人物传记**。

## 📁 文件结构

每个角色是一个**独立的 JSON 文件**，命名格式 `{名称}_{角色ID}.json`，可直接单独使用。

```
arknights-skill/
├── README.md
├── 阿米娅_char_002_amiya.json        # 罗德岛公开领袖
├── 凯尔希_char_003_kalts.json        # 罗德岛医疗负责人
├── W_char_113_cqbw.json              # 萨卡兹雇佣兵
├── 银灰_char_172_svrash.json         # 喀兰贸易总裁
├── 斯卡蒂_char_263_skadi.json        # 深海猎人
├── ... （共 492 个角色档案）
```

## 🔧 数据来源

每个角色的档案由以下数据组合而成：

| 来源 | 内容 |
|------|------|
| `character_table.json` | 角色名、种族、出身地 |
| `handbook_info_table.json` | **全部档案文本**（基础档案、客观履历、档案资料一~N、晋升记录、升变档案、临床诊断分析） |
| `charword_table.json` | **全部语音台词**（任命助理、交谈、晋升、信赖提升、作战等） |
| 手动编写 | 53 个关键角色的性格、人生经历、记忆、人际关系（基于游戏剧情） |

## 🚀 使用场景

- **AI 角色扮演** — 导入到 AI 聊天机器人作为角色卡（SillyTavern / Character Card）
- **同人创作** — 作为角色设定的权威参考
- **游戏研究** — 查阅角色的原始档案文本和语音
- **个人项目** — 独立 JSON，无依赖，直接嵌入任何项目
